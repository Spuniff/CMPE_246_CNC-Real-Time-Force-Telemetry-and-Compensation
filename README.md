# CMPE_246_CNC-Real-Time-Force-Telemetry-and-Compensation
CMPE_246 group project for creating CNC Real Time Force Telemetry and Compensation.
Goal is to create a device that will pause and lift a CNC machine when it detects that the instrument has deviated from its path.
After the lift the machine should recalculate proper path acounting for deviation and resume processing material.
